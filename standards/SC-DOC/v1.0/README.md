The Document Authoring Specification defines the complete authoring surface for all documents in the Structural Coherence ecosystem. It governs expression principles, content types, block and inline primitives, semantic containers, composition and nesting rules, format compatibility, class-specific content requirements, section templates, content ordering, and the pipeline interface between authoring and rendering.

The governing principle: **one content type, one expression.** Following the grammar makes the pipeline deterministic. Every content pattern that may appear in an SC-AS document is declared here.

SC-DOC-000001 supersedes SC-AUTH-000001, extending governance from the authoring layer alone to all nine cells of the document surface (Content, Structure, and Format across Presence, Connection, and Flow).
